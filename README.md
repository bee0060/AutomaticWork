###**EPAM_AUTO_RUN_ALL_SOFTWARE.bat**

是在EPAM任职期间，用于自动打开工作时所需几乎所有软件的一个批处理命令。
其中的软件包括：

1.outlook 

2.sublime text 

3.git bash 

4.chrome - localhost 

5.chrome - jira/pmc/google translate 

6.QQ 

并在打开全部软件后，执行fe live命令，用于启用http-server服务和watch服务。
